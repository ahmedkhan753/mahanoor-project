# Mahanoor's Romantic Website

This is a special website created for Mahanoor. It features festive animations, a romantic message, and an interactive proposal.

## How to Run

Since this is a frontend-only website, you can run it easily without any server:

1.  **Open the Project Folder**: Navigate to `d:\PROJECTS\mahanoor-project`.
2.  **Open the Website**: Double-click the `index.html` file. It will open in your default web browser (Chrome, Edge, etc.).
3.  **For Mobile**: If you want Mahanoor to see it on her mobile phone:
    *   The easiest way is to host it on a free service like [GitHub Pages](https://pages.github.com/) or [Netlify](https://www.netlify.com/).
    *   Alternatively, you can send the files to her phone and open `index.html` using a file manager app.

## Features
- Confetti explosion on load.
- Floating heart background.
- "No" button that moves when you try to click it (impossible to say no!).
- Full celebration when "Yes" is clicked.
